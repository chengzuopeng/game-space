# 🎮 Game Space / 游戏集合

这是一个精心打造的 HTML5 小游戏合集平台，**汇聚了来自开源社区的 60 款优秀小游戏**。我们致力于整理和分享这些可以在浏览器中直接运行的精彩内容，为您提供一个便捷的试玩与学习平台。

## ✨ 项目亮点

- **开源汇编**：本项目中的所有游戏资源均**收集自互联网开源项目及公开资源**，旨在致敬开源精神，方便开发者学习与参考。
- **丰富多样**：包含 60 款游戏，涵盖动作、益智、赛车、射击等多种题材。
- **即点即玩**：无需安装任何插件，纯现代浏览器支持。
- **响应式设计**：导航页适配桌面端与移动端设备。
- **纯静态架构**：无需后端数据库或复杂的构建流程，部署简单。

## 🚀 如何运行

由于这是一个纯静态项目，您可以通过以下几种方式运行：

### 方法 1：使用 VS Code Live Server (推荐)
1. 在 VS Code 中安装 **Live Server** 插件。
2. 右键点击 `index.html`。
3. 选择 "Open with Live Server"。

### 方法 2：使用 Python 快速启动
如果您安装了 Python，可以在终端中运行：

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```
然后访问 `http://localhost:8000`。

### 方法 3：直接打开
直接在浏览器中双击打开 `index.html` 文件。
> 注意：某些游戏可能依赖本地存储或特定加载方式，直接打开可能会受浏览器安全策略限制（CORS），推荐使用 Web 服务器方式运行。

## 🛠️ 技术栈

- **Frontend**: HTML5, CSS3, JavaScript
- **Game Engines**: 包含 Construct 2 导出项目、jQuery 插件及原生 Canvas 实现。
- **Styling**: 导航页使用原生 CSS Flexbox/Grid 布局与渐变设计，追求极简与美观。
